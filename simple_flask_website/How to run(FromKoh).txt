<How to run "AI Fact Checker">

1. Put the folder where appropriate on your laptop (it will put in TBC repo if you've just pulled from remote project folder)

2. Use terminal(Mac) or command prompt(Windows) to start. For windows, you can execute below commands in turn. 
------
cd <path to the folder>
python -m venv venv
. venv/Script/activate
pip install -r requirements.txt
flask run
------
